# Bookshop_management
This project is for Bookshop management.
It includes the following Use cases
1) Supplier
2) Employee
3) Books

1) Supplier Usecase:
  a) Add Supplier
  b) Remove Supplier
  c) Update Supplier Record

 2) Employee Usecase:
    a) Add Employee
    b) Remove Emlpoyee
    c) View Employee
    d) Search Employee
    e) Assign Manager Status
    f) Update Salary

4) Books
   a) Add Book
   b) Search Book
   c) Update price
